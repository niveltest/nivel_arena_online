# 音声演出の強化 (Audio Recommendations) Walkthrough

ゲーム内のアクションや状態変化に合わせて、よりきめ細やかな効果音（SE）が再生されるように実装を強化しました。

## 実施内容

### 1. 新規SEの定義とダミーファイルの作成

以下のSEを新たに追加し、既存の `bgm_battle.mp3` を複製したダミーファイルを配置しました（404エラーの防止とロジック確認のため）。

- `se_effect.mp3`: バフ/デバフ等の効果発動時
- `se_turn_start.mp3`: 自分のターンが回ってきた時
- `se_selection.mp3`: カード選択や防衛確認のダイアログが表示された時
- その他、欠落していた `se_destroy.mp3` などの定義済みSEもダミーで補完しました。

### 2. 再生ロジックの実装

`GameBoard.tsx` を修正し、以下のタイミングで自動的にSEが流れるようにしました。

- **アニメーション連動**:
  - ユニットが破壊された時 (`DESTROY` アニメーション)
  - カード効果が適用された時 (`EFFECT` アニメーション)
- **ターン開始通知**:
  - 手番が相手から自分に移った瞬間に通知音を再生。
- **操作ガイド音**:
  - 「カードを選択してください（SELECT_CARD）」や「ガードしますか？（GUARDIAN_INTERCEPT）」といった入力待ち状態になった際に、操作が必要であることを知らせる音を再生。

## 確認手順

1. **ターン交代**: 対戦中に相手のターンが終わり、自分のターンが始まった瞬間に音が鳴るか確認してください。
2. **ユニット破壊**: バトルや効果でユニットが消滅した際に音が鳴るか確認してください。
3. **効果発動**: バフなどの演出（浮遊テキスト等）が出た際に音が鳴るか確認してください。
4. **選択ダイアログ**: 効果によるカード選択画面や、相手の攻撃に対する防衛確認画面が出た際に音が鳴るか確認してください。

## 今後の対応

- 今回作成した `se_...` ファイルを本物の効果音ファイルに差し替えることで、演出のクオリティをさらに高めることができます。
- HP低下時の警告音など、より高度な演出は将来のアップデートで検討可能です。
