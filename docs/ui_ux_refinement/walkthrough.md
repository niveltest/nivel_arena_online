# 修正内容の確認: UI/UX の洗練 (Keyword Icons & Visual Effects)

ゲームの視認性と爽快感を向上させるための視覚効果を実装しました。

## 変更内容

### 1. キーワードアイコンの拡充 (`Card.tsx`)

カード右上のステータスアイコンを拡充し、以下のキーワードをアイコンで識別可能にしました：

- **【貫通】 (PENETRATION)**: 🏹 アイコン（オレンジ）
- **【略奪】 (LOOT)**: 💰 アイコン（イエロー）
- **【帰還】 (RECYCLE)**: ♻️ アイコン（グリーン）
- **【防壁】 (GUARDIAN)**: 🛡️ アイコン（ブルー）
- **【道連れ】 (DEATH_TOUCH)**: 💀 アイコン（ブラック）
- **攻撃不能**: 🚫 アイコン

### 2. プレイアブルガイドの実装 (`GameBoard.tsx`, `Card.tsx`)

プレイヤーが現在何ができるかを視覚的にガイドする機能を追加しました：

- **手札のグロー**: 現在のリーダーレベルとダメージゾーンから算出される制限サイズに収まるカードが、水色に発光（パルス）します。
- **ユニットの待機アニメーション**: 攻撃フェーズ中、まだ攻撃していない操作可能なユニットが小さく跳ねる（バウンス）アニメーションを行います。

### 3. フローティングテキストの強化 (`GameBoard.tsx`)

戦闘中に発生するテキスト表示をより詳細かつリッチにしました：

- **ダメージ**: `-数値`（赤）
- **回復**: `✚ Healing`（緑）
- **バフ**: `▲ Power +1000`（水色）
- **デバフ**: `▼ Power -1000`（赤）
- **レベルアップ**: `⭐ Level Up`（オレンジ）
- **破壊**: `💥 Destroyed`（紫）

### 4. アニメーションの追加 (`globals.css`)

- `animate-bounce-subtle`: 待機状態のユニット用の控えめな跳ねるアニメーションを追加しました。

## 検証結果

### 手動確認事項

- [x] キーワードを持つカード（例：「アバドン」の貫通など）に正しくアイコンが表示されること。
- [x] メインフェーズ中、コストの合計が上限を超えるカードが光らないこと。
- [x] 攻撃フェーズ中、まだ攻撃していないユニットがボヨンボヨンと動くこと。
- [x] ダメージ発生時やバフ発生時に、アイコン付きのカラーテキストが表示されること。
