# 実装計画: UI/UX の洗練 (視認性と爽快感の向上)

ゲームの視覚的なフィードバックを強化し、プレイ状況を直感的に把握できるようにします。

## ユーザーレビューが必要な項目

- **アイコンの選定**: 各キーワード（貫通、防御、略奪など）に使用するアイコンのデザイン。
- **アニメーションの激しさ**: 戦闘時のエフェクトの派手さの加減。

## 変更内容

### [Component] Card (`components/Card.tsx`)

- **キーワードアイコンの拡充**:
  - `PENETRATION` (貫通): 矢印または貫通をイメージするアイコン。
  - `LOOT` (略奪): 宝箱やコインを手に入れるアイコン。
  - `RECYCLE` (帰還): 矢印が戻るようなアイコン。
  - `DEATH_TOUCH` (道連れ): 骸骨アイコン（既存）の洗練。
- **バフ/デバフの強調**:
  - パワー値の変化をより目立たせ、変化があった際に数値がアニメーションするようにします。

### [Component] GameBoard (`components/GameBoard.tsx`)

- **フローティングテキストの強化**:
  - ダメージ（赤）、回復（緑）、バフ（青）など、種類に応じた色と動きのバリエーションを追加。
- **プレイアブルガイド**:
  - メインフェーズ中、コストを支払ってプレイ可能な手札に緩やかなグロー（発光）エフェクトを付与。
  - 攻撃フェーズ中、まだ攻撃していないユニットに待機アニメーションを付与。
- **ターゲットハイライト**:
  - アタッカー選択時、攻撃可能なスロットやリーダーを強調。

## 検証プラン

### 自動テスト

- ユニットテストの修正は不要。

### 手動検証

- ブラウザで対戦（CPU戦）を行い、以下の点を確認：
  - 各キーワードを持つカードに正しくアイコンが表示されているか。
  - ダメージ発生時に数値が飛び出しているか。
  - プレイ可能なカードが光っているか。
