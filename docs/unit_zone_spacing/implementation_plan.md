# 実装計画 - ユニットゾーンの間隔調整（垂直方向）

ユニットスロットを相手のユニットから 5% 遠ざけるように垂直方向の座標（`top`）を調整します。

## 変更内容

### [GameBoard.tsx](file:///c:/Users/worke/Antigravity/nivel_arena_online/components/GameBoard.tsx)

#### 1. 垂直座標の再計算

現行の `top: 15.0%` から、さらに 5% プレイヤー側（外側）へ移動させます。

- **垂直位置 (`top`)**: `15.0%` → `20.0%`
- **水平位置 (`left`)**: `32.0%`, `50.0%`, `68.0%` (現行の 18% 間隔を維持)

これにより、中央の境界線からスロットが遠ざかり、相手ユニットとの距離が広がります。水平方向の広がりは前回調整した値（18%間隔）に戻ります。

## 検証計画

### 手動確認

- 編集モードでリセットし、スロットがこれまでより手前（自分側）に配置されていることを確認。
- 相手側のスロットも同様に自分から遠ざかる方向に移動し、中央の空間が広がっていることを確認。
- プレイマット背景のユニット枠との位置関係に違和感がないか確認。
