# 実装計画 - ユニットゾーンの配置と公平性の確保

プレイマット画像の「Unit Zone」枠に収まるようサイズを調整し、かつ対戦相手のユニットと正確に正対（対向）するように水平・垂直方向の座標を統一します。

## 変更内容

### [GameBoard.tsx](file:///c:/Users/worke/Antigravity/nivel_arena_online/components/GameBoard.tsx)

#### 1. `PLAYMAT_CONFIGS.official` のユニット設定を定型化

ユーザーが手動で調整した不揃いな値を、均等なグリッド配置に修正します。

- **垂直位置 (`top`)**: 全てのユニットスロットを `15.0%` （センターライン付近）に統一します。
- **スケール (`scale`)**: 全てのスロットを `0.8` に統一し、枠内に収まりつつ視認性を確保します。
- **水平位置 (`left`)**:
  - `field1` (中央): `50.0%`
  - `field0` (左): `32.0%`
  - `field2` (右): `68.0%`
  （中心から左右に 18% ずつ等間隔に配置）

これにより、点対称（対角線）レイアウトにおいて、自分の `field0` と相手の `field0` が正確に盤面上で向かい合うようになります。

## 検証計画

### 手動確認

- 編集モードをオフにした状態で、ユニットが背景の「Unit Zone」の枠に綺麗に収まっているか確認。
- 相手側のユニット配置を確認し、自分のユニットの正面に相手のユニットが来ている（対向している）か確認。
- レイアウトエディタの「RESET」ボタンを押した際に、この新しい標準配置に戻ることを確認。
