# カードサイズ微調整計画

リーダー以外のカードをより縦長に見せるため、縦サイズを1%拡大します。

## 変更内容

### 1. カードコンポーネントの調整 (`Card.tsx`)

* [MODIFY] [Card.tsx](file:///c:/Users/worke/Antigravity/nivel_arena_online/components/Card.tsx)
  * `card.type !== 'LEADER'` の場合に、全体のスケールまたは高さを1%（縦方向）拡大するスタイルを追加します。
  * 計算上の複雑さを避けるため、`scale-y-[1.01]` を活用するか、高さを直接調整します。

## 検証プラン

* ビルドが通ることを確認。
* （ユーザー検証）
  * ユニットカードなどの縦がごく僅かに（1%）伸びていること。
  * リーダーカードのサイズが変わっていないこと。
