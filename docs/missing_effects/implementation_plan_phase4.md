# 実施計画 (Phase 4): 欠落・不十分なカード効果のブラッシュアップ

## 目的

本フェーズでは、これまでに実装した基本アクション (Phase 1-3) をさらに堅牢にし、公式ルールや特殊カードにより適合させます。

## 主な変更内容

### 1. ターン終了時処理の強化 (`Game.ts`)

- **歸還 (Recycle) の整理**: 現在 `nextPhase` 内で直接処理されている `isRecycle` のロジックを、より一般的なターン終了時処理 (End Turn Triggers) に統合します。
- **ターン終了時リセット**: ターン終了時にリセットされるキーワードやバフの処理を確実に実施します。

### 2. キーワード判定の堅牢化 (`Game.ts`)

- **表記ゆれ・テキスト判定の強化**: `hasKeyword` メソッドを改善し、`ARMED` (アームド)、`DEFENDER` (ディフェンダー) など、JSONの `keywords` だけでなく `text` 内の特定の日本語表記にもより正確に対応させます。

### 3. BUFF_SIZE とコスト制限の整合性

- **リーダーの覚醒反映**: ギルティ（ST02-001）などの覚醒によってサイズ制限が増える場合、その増分がカードのプレイ可否判定に常に正しく反映されているかを確認・修正します。

## 検証計画

- `ST02-001` (ギルティ) のサイズ+1が正しくコスト上限に反映されるか。
- `BT01-081` (チタンマターブーツ) が正常にトラッシュから手札に戻るか。
