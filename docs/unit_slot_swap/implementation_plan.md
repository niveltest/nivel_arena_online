# 実装計画 - 相手側ユニットスロットの反転（対向配置の正常化）

対角線（点対称）配置において、自分のスロット 0（左）と、相手の「対向ユニット」であるスロット 0 が盤面上で正確に向かい合うように修正します。

## 現状の課題

現在、相手側のプレイマット全体を 180 度回転させているため、相手のスロット 0（彼らにとっての左）は自分から見て「右」に位置しています。しかし、ゲームルール上の「対向ユニット」は自分と相手で同じインデックスのスロットを指すため、見た目上斜めに配置されているように見えてしまいます。

## 変更内容

### [GameBoard.tsx](file:///c:/Users/worke/Antigravity/nivel_arena_online/components/GameBoard.tsx)

#### 1. `PlaymatArea` におけるスロットマッピングの修正

相手側（`isOpponent: true`）のレンダリング時のみ、配置設定（座標）と実際のデータインデックスの紐付けを反転させます。

- **物理的な座標スロット 0 (左)**: 相手のデータインデックス `2` を表示
- **物理的な座標スロット 1 (中)**: 相手のデータインデックス `1` を表示
- **物理的な座標スロット 2 (右)**: 相手のデータインデックス `0` を表示

これにより、相手のスロット `0` は自分から見て「左」に配置され、自分のスロット `0` と正対するようになります。

## 検証計画

### 手動確認

- 自分のスロット 0（左）にユニットを出し、相手のスロット 0（自分から見て左）にユニットが出た際、一直線に並んでいるか確認。
- 相手のスロット 2（自分から見て右）が自分のスロット 2（右）と正対しているか確認。
- 「対向ユニットを選択」するような効果（あれば）において、見た目通りのユニットが対象になるか確認。
