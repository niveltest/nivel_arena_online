# モバイル表示最適化の完了

スマートフォン等の小型デバイスでのプレイ体験を向上させるための最適化が完了しました。
PCでの表示（1024px以上）は一切変更せず、画面幅が狭い場合のみ自動的に適用されます。

## 主な改善点

### 1. カードのレスポンシブ化

- 手札やフィールドのカードサイズをモバイル向けに縮小（`w-16 h-22`）。
- コスト（左上）、パワー（右上）、ヒット数（パワー下）のアイコンおよびフォントサイズを微調整し、小さいサイズでも視認性を維持。

### 2. ゲームボードのレイアウト調整

- **手札 (Hand)**: カード同士の重なりを調整し、画面幅に収まるように改善。
- **ヘッダー**: 部屋情報やフェイズバーをコンパクト化し、操作エリアを確保。
- **ボタン**: 「NEXT PHASE」ボタンなどのサイズをモバイル向けに最適化。

### 3. マリガン画面 (SelectionModal)

- カードを3列グリッドで表示し、一度に多くを確認できるように変更。
- ヘッダーや確認ボタンをコンパクトにし、スクロールの手間を軽減。
