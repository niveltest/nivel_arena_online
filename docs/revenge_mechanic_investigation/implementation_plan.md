# 道連れ（Revenge）機能の復旧

## 現状の確認結果

- `cards.json` 内を詳細に調査した結果、`BT02-024`（プリバティ‐ガバメントノーマル）などのテキストに「道連れ」の記述があることを確認しました。
- これらのカードには内部キーワードとして `DEATH_TOUCH` が設定されています。
- 以前の調査で見落としていた点をお詫びします。

## 提案される変更

- **components/Card.tsx**:
  - `keywords.includes('DEATH_TOUCH')` の条件でスカルアイコン（💀）を表示するように復旧します。
  - 以前削除した `TARGET` および `SNIPER` については、データ上に存在しないことが改めて確認されたため、非表示のまま維持します。

## 検証計画

- 開発環境で `BT02-024` カードを表示し、アイコンが正しく表示されることを目視確認します。
