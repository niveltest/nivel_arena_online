# 道連れ（Revenge）機能の復旧の完了報告

「道連れ」機能がデータ上に存在することを確認し、UI上のアイコン表示を復旧しました。

## 修正内容

- **components/Card.tsx**:
  - 以前削除した「道連れ」アイコン（💀）を復旧しました。
  - 判定条件として、内部キーワードである `DEATH_TOUCH` およびテキスト内の「道連れ」という文字列を追加しました。
  - これにより、`BT02-024`（プリバティ）などのカードで正しくアイコンが表示されるようになります。

## 以前の修正との違い

- 「狙撃（SNIPER）」および「集中攻撃（TARGET）」については、引き続きデータ上に見当たらないため、非表示のままとしています。

## 検証結果

- `BT02-024` のデータ構成に基づき、`DEATH_TOUCH` キーワードを持つカードでアイコンが表示されるロジックであることを確認しました。
