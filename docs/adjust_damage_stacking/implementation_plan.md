# ダメージカード重なり方向の反映計画

ダメージゾーンのカードが蓄積する際、一枚目の位置を動かさず、指定された方向へ重なるように修正します。

## 変更内容

### 1. ダメージゾーンの配置ロジック修正 (`GameBoard.tsx`)

* [MODIFY] [GameBoard.tsx](file:///c:/Users/worke/Antigravity/nivel_arena_online/components/GameBoard.tsx)
  * **アンカーの固定**: 各ダメージカードを包むコンテナの `justify-center` を `justify-start` に変更し、新しいカードが追加されても一枚目のカードの位置がずれないようにします。
  * **重なり方向の制御**:
    * 自分側 (`isOpponent === false`): デフォルトのままでもコンテナの回転がないため、右方向へ重なっていきます。
    * 相手側 (`isOpponent === true`): 親要素に `rotate-180` がかかっているため、`justify-start` で右端（自分から見て）に一枚目が配置され、追加されるカードは左方向へ展開されます。
  * **CSSの調整**: `margin-left` を調整し、適切な重なり具合（カードの一部が見える状態）にします。

## 検証プラン

* ビルドが通ることを確認。
* （ユーザー検証）
  * 自分のダメージゾーンにカードが増えた際、一枚目が動かず、右側へ重なっていくこと。
  * 相手のダメージゾーンにカードが増えた際、一枚目が動かず、左側へ重なっていくこと。
