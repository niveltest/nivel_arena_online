# 公式Q&A裁定に基づく修正と洗練：完了報告

公式Q&Aにある詳細な裁定やエッジケースの挙動を調査し、ゲームエンジンに反映させました。

## 実施内容

### 1. 「デュエリスト」(DUELIST) の制限強化

- **変更内容**: アタッカーが `DUELIST` を持ち、かつ正面に防衛ユニットがいる場合、相手は隣接レーンの 〈ガーディアン〉(防壁) で割り込むことができないように修正しました。

### 2. 〈エントリー〉(ON_ENTRY) の導入と全領域対応

- **変更内容**: カード配置時のトリガーを `ON_ENTRY` に整理し、手札からだけでなく、他の領域（トラッシュ等）からの登場時にも正しく発動できるようにしました。

### 3. ルール破壊（0パワー）の優先

- **変更内容**: パワーが 0 以下になったことによる破壊はルールの強制執行であるため、「効果で破壊されない」耐性を無視してトラッシュに送られるように修正しました。

### 4. 相手ユニットへのアイテム装備

- **変更内容**: 自分だけでなく相手のユニットスロットもアイテムの装備対象に選べるようにしました。デバフアイテムの押し付けなどが可能になります。

### 5. キーワード値の重複解決

- **変更内容**: 同一のカードが複数の数値付きキーワードを持つ場合、加算ではなく公式裁定通り「最大値」を採用するように修正しました。

## 検証結果

- `DUELIST` 持ち攻撃時に条件を満たせば隣接防壁がスキップされることを確認。
- パワー0の無敵ユニットがルールに従い破壊されることを確認。
- 敵陣へのアイテム装備と効果発動が正しく行われることを確認。
