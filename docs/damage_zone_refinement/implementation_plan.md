# 実装計画 - ダメージゾーンの配置と重なり具合の修正

提供された画像に基づき、ダメージゾーンのカード表示が背景のプレイマット画像のスロットに完璧に重なるよう調整します。

## 現状の課題

- 現在、カードの表示が大きすぎて背景のスロット（斜めのライン）からはみ出している。
- カード間の重なりが不十分で、10枚並んだ際に背景のデザインと一致していない。

## 変更内容

### [GameBoard.tsx](file:///c:/Users/worke/Antigravity/nivel_arena_online/components/GameBoard.tsx)

#### 1. カードの表示形式の調整

- `DraggableZone` 内のカードスケールを、背景のスロット幅に収まるようさらに縮小します。
- `marginLeft` の値を調整し、背景の斜めのラインとカードの左端が揃うように微調整します。
- 背景の斜線に合わせるため、カード自体に僅かな `skew`（傾斜）を適用することを検討します。

#### 2. デフォルト座標の調整

- `PLAYMAT_CONFIGS.official` の `damage` ゾーンの位置 (`top`, `left`) と全体の `scale` を、実際のプレイマット画像のダメージエリアに重なるよう再調整します。

## 検証計画

### 手動確認

- ダメージゾーン背景の斜線10本と、カードの左端が一致しているか確認。
- 複数枚のカードが重なった際、背景のスロットを埋めていくように見えるか確認。
- カードが大きすぎず、プレイマット上の「DAMAGE」の枠に収まっているか確認。
