# カードUI微調整計画

カード上の表示項目を整理し、視認性を高めます。

## 変更内容

### 1. カード表示レイアウトの修正 (`Card.tsx`)

* [MODIFY] [Card.tsx](file:///c:/Users/worke/Antigravity/nivel_arena_online/components/Card.tsx)
  * **所属 (affiliation)**: カードの上部中央に配置します。また、「/」以降の文字列を削除し、「ベース」や「エフェクト」などの主要な情報のみを表示するようにします。
  * **パワー (power)**: カードの右上に移動し、視覚的に目立たせます。
  * `minimal` モードではない場合のみ、これらの調整を適用します。
