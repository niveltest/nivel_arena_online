# 今後の開発ロードマップ：提案

現在の Nivel Arena Online は、基本ロジックとカードデータの自動パース、そして基本的な UI が揃った状態です。さらに完成度を高めるための次のステップとして、以下の3つの方向性を提案します。

## 1. AI (CPU) の戦略的強化

現在の AI は基本的なプレイ（出せるカードを出す、正面を攻撃する等）のみを行います。これをより「賢く」することで、一人プレイの満足度を高めます。

- **手札管理**: 将来のターンのために強力なユニットを温存する、コストに合わせたマリガン。
- **ターゲット選択**: 相手の強力なユニットを優先的に除去する、あるいはリーダーへのトドメを優先する判断。
- **防御判断**: どのユニットを犠牲にしてライフを守るか、あるいは無視してカウンターを狙うかの最適化。

## 2. 特殊ルールとキーワードの深化 (UNION / LEGION)

カードデータ内に存在するが、まだ処理ロジックが単純な、あるいは未実装の高度なルールを完全にサポートします。

- **ユニオン (UNION)**: 素材ユニットの能力（キーワードや効果）をベースユニットが一定期間継承する仕組みの完全実装。
- **レギオン (LEGION)**: 特定の構成（レギオンメンバー）が揃った時に発動するパッシブ/アクティブ効果の実装。
- **高度なスタック処理**: 複数の効果が同時に発動した際の解決順序の制御。

## 3. UI/UX の洗練と視覚効果 (WoW Factor)

プレイヤーがゲームの状況を一目で把握でき、かつ爽快感を感じられるように UI をアップグレードします。

- **ステータス可視化**: バフ/デバフによるパワーの変化を数値の色（緑/赤）やアイコンで表示。
- **戦闘アニメーション**: ダメージの飛び出し（フローティングテキスト）、貫通や略奪発生時の専用エフェクト。
- **キーワードアイコン**: カード上に「貫通」「防壁」「ディフェンダー」などのアイコンを常時表示し、把握しやすくする。

---

## 4. オンライン対戦環境の構築とテスト

ローカル環境を超えて、どこでも誰とでも遊べる環境を構築します。

- **デプロイ設定**: Render (Server) や Vercel (Client/Frontend) への自動デプロイ設定の追加。
- **環境のオンライン最適化**: ローカルIPのハードコードを排除し、環境変数による動的な接続先切り替えの実装。
- **テストプレイ支援**: トンネリングツール (ngrok) などを用いて、最小限の手間で外部の友人を招待できる手順の整備。
- **セッション維持・再接続**: モバイル回線など不安定な環境でもゲームを継続できるよう、再接続ロジックの堅牢化。
