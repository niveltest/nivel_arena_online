# ウォークスルー - 自分・相手エリアのサイズ同期修正

自分側と相手側でカードサイズが異なって表示される問題を修正し、全てのゾーンで見た目を統一しました。

## 変更内容

### [Component] [GameBoard.tsx](file:///c:/Users/worke/Antigravity/nivel_arena_online/components/GameBoard.tsx)

- **サイズの同期 (統一スケール適用)**:
  - 相手側のゾーンが小さく表示されていた原因となる固定コンテナサイズや重複したスケーリングを整理しました。
  - 全ての周辺ゾーン（ダメージ、デッキ、スキル、トラッシュ）のスケールを、自分・相手ともに `0.8` に統一しました。
- **配置ロジックの標準化**:
  - `DraggableZone` 内部のレイアウトを自分と相手で共通化し、180度回転の影響を受けずに同じ物理サイズで表示されるようにしました。
- **重なりの再微調整**:
  - カードサイズが本来の大きさに戻ったため、ダメージゾーンのカードの重なり間隔（`margin-left`）を `-6.5rem` に戻し、密集しすぎないようにしました。

## 検証結果

- **視認性の統一**: 自分の手前にあるデッキやダメージゾーンと、相手側の奥にある各ゾーンが、全く同じカードサイズで表示されることを確認しました。
- **レイアウトの整合性**: 鏡合わせ（対角線対称）の配置を維持しつつ、視覚的な大きさが一致していることを検証しました。
