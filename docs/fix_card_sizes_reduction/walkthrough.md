# ウォークスルー - 全カードサイズの20%縮小

フィードバックに基づき、全てのプレイマットゾーンにおいてカードサイズを一律20%縮小し、レイアウトにゆとりを持たせました。

## 変更内容

### [Component] [GameBoard.tsx](file:///c:/Users/worke/Antigravity/nivel_arena_online/components/GameBoard.tsx)

- **一律スケールダウン**:
  - `official` プレイマット設定の全ゾーン（フィールド、ダメージ、デッキ、スキル、トラッシュ）のスケールを、従来の `0.8` から `0.64` (20%減) に変更しました。
- **リーダーの縮小**:
  - リーダーカードのスケールも `0.7` から `0.56` (20%減) に縮小し、全体のバランスを整えました。
- **配置の最適化**:
  - カードサイズが小さくなったのに合わせ、ダメージゾーンのカードの重なり間隔（`margin-left`）を `-6.5rem` から `-5rem` に調整しました。これにより、小さくなったカード同士が自然に重なって見えるようになります。

## 検証結果

- **視認性**: カードが一回り小さくなり、プレイマットの各枠に対して適切な余白が生まれ、画面全体がスッキリしたことを確認しました。
- **重なり**: 小さくなったダメージカードが適切に整列し、綺麗にスタックされていることを確認しました。
