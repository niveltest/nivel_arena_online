# 音量調整UIの実装計画

ユーザーがブラウザ上でBGMとSEの音量を自由に変更できるように、設定モーダルとスライダーを追加します。

## 変更内容

### 1. 新規コンポーネントの作成

- **`components/AudioSettingsModal.tsx` [NEW]**
  - BGMとSEの音量を調整するスライダーを提供。
  - `audioManager.getVolume()` / `setVolume()` を使用してリアルタイムに値を反映。
  - デザインは既存のUIに合わせ、半透明のオーバーレイとモダンなスライダーを採用。

### 2. 既存画面への統合

- **`components/Lobby.tsx` [MODIFY]**
  - 画面右上に歯車アイコン（設定ボタン）を追加。
  - クリック時に `AudioSettingsModal` を表示。
- **`components/GameBoard.tsx` [MODIFY]**
  - 画面右上（または適切な空きスペース）に設定ボタンを追加。
  - ロビーと同様にモーダルを表示可能にする。

### 3. スタイリング

- Tailwind CSS を使用して、ゲーミングデバイスのようなクールなデザインにします。
- アイコンにはシンプルなテキスト（⚙️）またはライブラリを使用（今回は依存を増やさないため絵文字かSVGパスで対応）。

## 検証計画

### 手動確認

1. ロビーで設定ボタンを押し、スライダーを動かしてBGM音量が変わることを確認。
2. そのままゲームを開始（またはリロード）し、設定した音量が維持されていることを確認。
3. 対戦画面でも同様にSEの音量が変化することを確認。
