# 修正内容の確認 - レイアウトエディタのサイズ調整機能

レイアウトエディタに各ゾーンのサイズ（スケール）を自由に変更できる機能を追加しました。

## 変更内容

### [GameBoard.tsx](file:///c:/Users/worke/Antigravity/nivel_arena_online/components/GameBoard.tsx)

#### 1. インタラクティブなサイズ調整

- `DraggableZone` に黄色い丸型のリサイズハンドルを追加しました。
- ハンドルをドラッグすることで、ゾーンの中心からの距離に応じて `scale` がリアルタイムに変化します。
- スケール値は自動的に `onLayoutChange` を通じて `customLayout` に反映されます。

#### 2. 表示の一貫性

- ゾーンの左上に現在のスケール値（SC: 0.XX）が表示されるようにし、微調整をしやすくしました。
- 従来の `PlaymatArea` で行っていた CSS（CSS変数）によるスケーリングを廃止し、`DraggableZone` の `transform` に統合することで、二重にサイズが変わる問題を回避しました。

## 検証結果

- [x] 編集モードでリサイズハンドルが表示されることを確認。
- [x] ハンドルのドラッグにより、リーダー、ユニットスロット、山札などのサイズがスムーズに変わることを確認。
