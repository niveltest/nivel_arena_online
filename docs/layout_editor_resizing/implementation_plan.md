# 実装計画 - レイアウトエディタのサイズ調整機能

レイアウトエディタモードにおいて、各ゾーンのサイズ（スケール）をマウス操作で変更できるようにします。

## 概要

各 `DraggableZone` の右下にリサイズハンドルを表示し、それをドラッグすることで `scale` 値を動的に更新します。

## 変更内容

### [GameBoard.tsx](file:///c:/Users/worke\Antigravity\nivel_arena_online\components\GameBoard.tsx)

#### 1. `DraggableZone` コンポーネントの更新

- リサイズハンドル（右下の小さな四角形）を追加。
- ハンドルは `isEditMode` の時のみ表示。
- ハンドルの `onDrag` イベントで、ゾーンの中心からの距離に基づき新たな `scale` を計算。
- `onLayoutChange` を呼び出して `scale` を永続化（一時的状態）。

#### 2. スケール計算ロジック

- 基準となるサイズ（例：100px）に対して、ドラッグ後のハンドル位置がどの程度の比率になるかを算出します。
- 最小スケール（0.1）と最大スケール（3.0）の制限を設けます。

## 検証計画

### 手動確認

- エディタモードで右下のハンドルをドラッグし、サイズがリアルタイムに変わるか確認。
- サイズ変更後、JSON エクスポートに新しい `scale` 値が反映されているか確認。
- 「RESET」で元のスケールに戻るか確認。
