# 修正内容の確認：先攻・後攻ラベルの追加

## 概要

対戦開始時および対戦中に、自分が「先攻」か「後攻」かを一目で判断できるようにUIを改善しました。

## 修正内容

### 1. マリガン画面 (初期手札交換)

- マリガン画面のタイトルの右側に、「先攻」または「後攻」の大きなバッジが表示されます。
- **先攻**: 赤色のバッジ
- **後攻**: 青色のバッジ

### 2. 対戦中 (ステータスバー)

- 画面上部のステータスバー（Turn数「T-1」の横）に、常時自分の手番順が表示されます。
- これにより、ターンが経過しても自分がどちらだったかを忘れずに済みます。

## 確認手順

1. アプリを起動し、対戦を開始します（CPU戦でも確認可能です）。
2. **マリガン画面**:
   - ステップ1の画面で、タイトル「🔄 マリガン [手札]」の横にバッジが出ていることを確認してください。
3. **対戦開始後**:
   - 画面上部、左側の「T-1」という表示の右側に、同様のバッジが残っていることを確認してください。

## 補足

- この判定はサーバーから送られる `turnPlayerId` と現在の `turnCount` に基づいて自動的に計算されます。
- 途中で再接続（リロード）しても、正しく現在の状態が表示されます。
