# ゾーンサイズ修正のウォークスルー

## 修正内容

### ゾーンサイズの拡大

`GameBoard.tsx` において、以下のゾーンのサイズを `w-[10cqw]` から `w-[13cqw]` に変更しました。

- デッキゾーン (Deck)
- トラッシュゾーン (Trash)
- スキルゾーン (Skill Zone)

これにより、これらのゾーンが標準のユニットスロットと同じサイズになり、プレイマット背景の枠線とサイズが一致するようになりました。

### スキルゾーンのテキスト調整

スキルゾーンのラベルテキストサイズを `text-[1cqw]` から `text-[1.2cqw]` に調整し、拡大されたスロットとのバランスを取りました。

## 確認事項

- プレイマット上の各ゾーンの枠と、実際のカードやプレースホルダーのサイズが一致していること。
- レイアウトが崩れていないこと。
