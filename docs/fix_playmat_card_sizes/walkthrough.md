# プレイマットスケーリング修正のウォークスルー

## 修正内容

### プレイマットのサイズ制限の復元

`GameBoard.tsx` において、`playmat-canvas` のクラスを修正しました。

```tsx
// 変更前
<div className="relative w-full aspect-video ...">

// 変更後
<div className="relative h-full w-auto max-w-full max-h-full aspect-video ...">
```

これにより、プレイマットは親コンテナ（画面の残り領域）の「高さ」を基準にしつつ、`aspect-video` (16:9) を維持し、かつ幅が画面幅を超えない (`max-w-full`) ように自動調整されます。
結果として、画面サイズに関わらず、プレイマット全体が見切れることなく表示され、上下または左右に適切な余白（黒帯）が生じるようになります。

### フェーズ表示

フェーズ表示等のトップバーは変更していないため、画面上部に維持されています。

## 確認事項

- プレイマットが画面からはみ出さずに全体が表示されていること。
- カードサイズが引き続き相対的に（`cqw`により）正しい比率で表示されていること。
