# カードアスペクト比調整のウォークスルー

## 修正内容

### カードの縦長化 (1%)

`GameBoard.tsx` において、以下のゾーンのカードスロットのアスペクト比を `5/7` (約0.714) から `500/707` (約0.707) に変更しました。これにより、縦方向の比率が約1%増加しました。

- フィールドスロット (`renderSlot`)
- デッキゾーン (Deck)
- トラッシュゾーン (Trash)
- スキルゾーン (Skill Zone)

### 除外された項目

以下の要素はアスペクト比 `5/7` のまま維持しました。

- ダメージゾーン (Damage)
- リーダーカード (Leader)

## 結果

カードが以前よりもごくわずかに縦に長くなり、指定された要件を満たしています。
