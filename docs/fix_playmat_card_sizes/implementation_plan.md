# ゾーンサイズ調整の実装計画

## 問題

デッキ、トラッシュ、スキルゾーンのカード表示サイズ (`w-[10cqw]`) が、プレイマットの背景に描かれた枠（標準カードサイズ想定）よりも小さく、フィットしていない。

## 解決策

これらのゾーンのサイズ指定を、標準のユニットスロットと同じ `w-[13cqw]` に変更し、プレイマットの枠に合わせる。

### [MODIFY] [GameBoard.tsx](file:///c:/Users/worke/Antigravity/nivel_arena_online/components/GameBoard.tsx)

- `deck` ゾーンのクラス: `w-[10cqw]` -> `w-[13cqw]`
- `trash` ゾーンのクラス: `w-[10cqw]` -> `w-[13cqw]`
- `skill` ゾーンのクラス: `w-[10cqw]` -> `w-[13cqw]`
- スキルゾーンのテキストサイズ調整: `text-[1cqw]` -> `text-[1.2cqw]` (バランス調整)

## 検証

- 背景の枠線と実際のカード/プレースホルダーのサイズが一致することを確認する。
