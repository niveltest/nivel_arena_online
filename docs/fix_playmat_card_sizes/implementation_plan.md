# カードアスペクト比調整の実装計画

## 目標

リーダーカードとダメージカードを除くカードのサイズを、縦に1%大きくする。

## 実装方針

現在の `aspect-[5/7]` (約1:1.4) をベースに、高さを1%増やすアスペクト比に変更する。

- 現在の比率: 幅/高さ = 5/7 ≒ 0.7142
- 新しい高さ: 7 * 1.01 = 7.07
- 新しい比率: 幅/高さ = 5/7.07 ≒ 0.7072
- Tailwindの任意値 `aspect-[500/707]` を使用する。

### [MODIFY] [GameBoard.tsx](file:///c:/Users/worke/Antigravity/nivel_arena_online/components/GameBoard.tsx)

以下の箇所の `aspect-[5/7]` を `aspect-[500/707]` に置換する。

1. **renderSlot**: フィールド上のユニットカード
2. **deck**: デッキ置き場
3. **trash**: トラッシュ置き場
4. **skill**: スキルゾーン

**除外対象 (変更しない):**

- **damage**: ダメージゾーン (`aspect-[5/7]` のまま維持)
- **leader**: リーダーカード (個別の設定を使用しているため対象外)

## 検証

- 指定したゾーンのカードがわずかに縦長になっていること。
- ダメージゾーンとリーダーカードのサイズが変わっていないこと。
