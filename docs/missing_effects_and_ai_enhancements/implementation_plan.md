# 実装計画: 欠落しているカード効果の実装 (Phase 3) および AI 戦略の強化

## 目標

欠落していたカード効果（全対象バウンス、全対象ヒットバフ、装備数参照条件）を実装し、AIがこれらの効果やスキルをより戦略的に使用できるように強化します。

## 変更内容

### 1. カード効果の実装 (Game.ts)

- `BOUNCE_UNIT` の `ALL_ENEMIES` ターゲット対応
- `BUFF_HIT` の `ALL_ALLIES` ターゲット対応
- `OPPOSING_HIT_LE_ARMED_COUNT` 条件の実装

### 2. AI 戦略の強化 (AIPlayer.ts)

- **スキル使用ロジック**: スキルの有用性（例：ユニットがいない時のバフを控える）を判定し、コストパフォーマンスに基づいて使用を検討。
- **選択ロジック**: バフは自軍の有力ユニットに、デバフは敵軍の脅威ユニットに優先的に適用。
- **コードクリーンアップ**: 冗長なメソッドやスイッチケースを削除し、一貫性を向上。

## 検証計画

- `unitTestPhase3.ts`: 新規カード効果のロジック検証
- `unitTestAISkill.ts`: AIのスキル使用およびターゲット選定ロジックの検証
