# ウォークスルー: UI/UX および視覚効果の強化

## 概要

ゲームの視覚的なフィードバックを強化し、戦闘の状況やユニットの状態を一目で把握できるようにしました。また、現状のカードデータに存在しない不要な要素を整理しました。

## 変更内容

### 1. カードの状態可視化 (`components/Card.tsx`)

- **動的な数値カラー**:
  - **緑**: バフ適用中 (`tempPowerBuff > 0`)。
  - **赤**: デバフ適用中 (`tempPowerDebuff > 0`)。
- **バフ・ハイライト**:
  - パワーが増加しているユニットに、グリーンの外光（Glow）とリングエフェクトを追加しました。
- **アイコンの整理**:
  - 現状のカードデータで使用されていないため、混乱を避けるために `TARGET`(集中攻撃) および `SNIPER`(狙撃) アイコンは実装から除外（削除）しました。

### 2. 戦闘・効果アニメーション (`components/GameBoard.tsx`)

- **フローティングテキスト・システム**:
  - ダメージ（赤）、回復（緑）、バフ（青/オレンジ）、スタン（紫）などの数値やテキストが、対象のユニットやリーダーの上に飛び出すように実装しました。
  - 各プレイマット（Official/Mermaid/Cyber）のレイアウトに対応し、適切な位置に表示されます。

### 3. ゲームロジックとの統合 (`server/Game.ts`)

- サーバー側で `EFFECT` アニメーションタイプを新設し、ダメージ以外の各種効果（回復、バフ等）をクライアントに通知する仕組みを構築しました。

## 確認方法

1. CPU対戦を開始します。
2. スキルカードやカード効果を使用してバフ/デバフを適用し、数値の色とグロー効果を確認します。
3. 攻撃や回復を行い、各ユニットの上に適切な色のテキストが表示されることを確認します。
