# ウォークスルー - 全ゾーンのカードサイズ統一

ダメージゾーン、デッキ、スキル、トラッシュの各ゾーンのカードサイズを、フィールド（ユニットゾーン）と同じサイズに統一しました。

## 変更内容

### [Component] [GameBoard.tsx](file:///c:/Users/worke/Antigravity/nivel_arena_online/components/GameBoard.tsx)

- **スケールの統一**:
  - `official` プレイマット設定において、全てのゾーン（フィールド、リーダー、ダメージ、デッキ、トラッシュ、スキル）のスケールを `0.8` 前後に統一しました。
- **コンテナサイズの標準化**:
  - デッキ、トラッシュ、スキルゾーンのプレースホルダーサイズを `w-16 h-24` から `w-28 h-40`（ユニットカードの標準サイズ）に拡大しました。
  - 内部で個別に行っていたスケール調整（`scale-[0.8]` や `scale-[0.5]` など）を削除し、全てのカードが本来のサイズで表示されるようにしました。
- **ダメージゾーンの調整**:
  - カードサイズが大きくなったのに合わせ、カードが綺麗に重なるように `margin-left` の間隔を `-4.5rem` から `-6.5rem` に広げました。
- **視認性の向上**:
  - 全てのゾーンでカードが大きく表示されるようなり、対戦中の状況（残りデッキ枚数やトラッシュの内容など）が把握しやすくなりました。

## 検証結果

- **視認性**: 全てのゾーンでカードがユニットゾーンと同じ物理サイズで表示されることを確認しました。
- **配置**: カードサイズ拡大後も、各ゾーンがプレイマットの枠内に適切に収まっていることを確認しました。
