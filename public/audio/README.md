# Audio Setup Guide

このフォルダに音声ファイルを配置することで、ゲーム内のBGMやSEをカスタマイズできます。
以下のファイル名で保存してください。

## 必須フォーマット

- **形式**: MP3推奨 (ブラウザ互換性のため)
- **場所**: このフォルダ (`public/audio/`) 直下

## ファイル名一覧

### BGM

| ファイル名        | 用途                 |
| :---------------- | :------------------- |
| `bgm_battle.mp3`  | バトル中の背景音楽   |

### SE (効果音)

| ファイル名            | 用途                           |
| :-------------------- | :----------------------------- |
| `se_draw.mp3`         | カードをドローした時           |
| `se_play_card.mp3`    | ユニットやカードをプレイした時 |
| `se_attack_start.mp3` | 攻撃宣言時 (発砲音など)        |
| `se_attack_hit.mp3`   | 攻撃がヒットした時             |
| `se_damage.mp3`       | ダメージを受けた時             |
| `se_destroy.mp3`      | ユニットが破壊された時         |
| `se_levelup.mp3`      | レベルアップ / 覚醒時          |
| `se_victory.mp3`      | 勝利時                         |
| `se_defeat.mp3`       | 敗北時                         |

---
**注意**: 著作権のある素材を使用する場合は、個人の利用範囲内（または配信ガイドライン等）で適切に取り扱ってください。
