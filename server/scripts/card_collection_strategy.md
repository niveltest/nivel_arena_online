# スターターデッキカード情報収集スクリプト

このスクリプトは公式サイトから段階的にカード情報を収集するためのメモです。

## 収集方針

レート制限を回避するため、以下の戦略を採用します：

1. **既知のカードから推測**: 既存のBT02カードのパターンから、スターターデッキのカード構成を推測
2. **段階的な収集**: 一度に大量のリクエストを送らず、時間を置いて収集
3. **手動補完**: 自動取得できない情報は後で手動で追加

## スターターデッキの構成（推定）

各スターターデッキには通常以下が含まれます：

- リーダーカード: 1枚
- ユニットカード: 約15-20枚
- スキルカード: 約5-10枚
- アイテムカード: 約3-5枚

合計: 約25-35枚/デッキ

## 次のステップ

1. 60秒待機してレート制限をクリア
2. ブラウザサブエージェントで再試行
3. 成功したカードから順次追加
