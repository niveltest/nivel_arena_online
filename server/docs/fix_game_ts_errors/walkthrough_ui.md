# DeckBuilder UI 改善の確認

デッキ構築画面（Deck Builder）において、カードの視認性を向上させ、表示の崩れを修正しました。

## 修正内容

### 1. カードの見切れ修正

- [DeckBuilder.tsx](file:///c:/Users/worke/Antigravity/nivel_arena_online/components/DeckBuilder.tsx) の各グリッドコンテナにパディング (`p-4`) を追加しました。これにより、カードの左上にはみ出しているコストバッジが縁で見切れる問題が解消されました。
- コンテナにうっすらと背景色 (`bg-white/5`) と枠線を追加し、境界を明確にしました。

### 2. 余白と配置の最適化

- **現在のデッキ（右側）**: 従来のネガティブマージンによる重ね合わせを廃止し、`gap-y-12` を使用した整列グリッドに変更しました。これにより、カード同士が適切に重なりつつも、個々のカードが認識しやすくなりました。
- **カード一覧（左側）**: `gap-4` に広げることで、カード同士の密着を避け、すっきりとした見た目にしました。

### 3. ホバーエフェクトの調整

- [Card.tsx](file:///c:/Users/worke/Antigravity/nivel_arena_online/components/Card.tsx) のホバー時の拡大率を `1.5倍` から `1.1倍` に変更しました。これにより、特定のカードを見た際に周囲のカードが極端に隠れてしまうストレスを軽減しました。
